1
Exercice 1 — Quelle commande Git utiliser ?

Pour chaque situation, choisis la bonne commande Git à taper dans le terminal.

Tu veux vérifier si Git est installé sur ton PC. Quelle commande tapes-tu ?
Tu veux créer un nouveau dépôt Git dans ton dossier de projet. Quelle commande tapes-tu ?
Terminal
$ git status On branch main modified: index.html modified: style.css
Tu vois que deux fichiers ont été modifiés. Tu veux tout préparer pour un commit d'un seul coup. Quelle commande utilises-tu ?
Tu veux télécharger pour la première fois un projet depuis GitHub sur ton PC. Quelle commande utilises-tu ?
Tu travailles sur un projet en équipe. Quelqu'un a fait des modifications sur GitHub. Tu veux récupérer ses changements. Quelle commande utilises-tu ?
Tu as fait git add .. Maintenant tu veux créer la sauvegarde avec le message "Ajout du formulaire". Quelle commande ?
