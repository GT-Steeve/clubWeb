Exercice 2 — Trouve l'erreur dans chaque bloc JavaScript

Chaque bloc ci-dessous contient une erreur. Clique sur la correction qui convient !

Erreur 1 — la déclaration de variable est incorrecte :
variable age = 14; // ← ce mot n'existe pas en JS moderne
Erreur 2 — le test de comparaison est mal écrit :
if (score = 10) { console.log("Gagné !"); } // ← = assigne, ne compare pas !
Erreur 3 — il manque quelque chose dans la boucle for :
for (let i = 0; i < 5) { console.log(i); } // ← boucle infinie !
Erreur 4 — la fonction ne renvoie rien :
function double(n) { n * 2; // ← le résultat est calculé mais perdu ! } console.log(double(5)); // affiche undefined
Erreur 5 — l'accès au tableau est incorrect :
const couleurs = ["rouge", "vert", "bleu"]; console.log(couleurs[3]); // ← affiche undefined ! // On voulait afficher "bleu"
