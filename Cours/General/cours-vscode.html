C'est quoi un IDE ?

Un IDE (Integrated Development Environment, ou Environnement de Développement Intégré) est un logiciel spécialement conçu pour écrire du code. C'est comme un traitement de texte (Word, LibreOffice…), mais en beaucoup plus puissant pour les développeurs !

Un IDE t'aide à :

✏️ Écrire du code avec de la coloration et de l'autocomplétion
🔍 Détecter les erreurs avant même de lancer ton programme
📁 Organiser tous tes fichiers dans un seul endroit
🚀 Lancer et tester ton code directement depuis l'éditeur
💡 Sans IDE, tu écrirais ton code dans le Bloc-notes — ça marche, mais tu n'aurais aucune aide. L'IDE, c'est le filet de sécurité du développeur !
★
Pourquoi choisir VS Code ?

Il existe beaucoup d'IDEs (IntelliJ, Sublime Text, Atom…), mais Visual Studio Code (ou VS Code) est de loin le plus utilisé au monde. Voici pourquoi :

1
Gratuit et open source Développé par Microsoft, VS Code est entièrement gratuit, pour toujours.
2
Compatible avec tous les langages HTML, CSS, JavaScript, Python, PHP, Java… il gère tout !
3
Des milliers d'extensions Tu peux ajouter des fonctionnalités en un clic grâce à la marketplace d'extensions.
4
Léger et rapide Il démarre en quelques secondes et tourne bien même sur des machines modestes.
⬇
Télécharger & installer VS Code

L'installation est simple et rapide. Suis ces étapes :

1
Va sur le site officiel Ouvre ton navigateur et rends-toi sur code.visualstudio.com
2
Clique sur le bouton de téléchargement Le site détecte automatiquement ton système (Windows, Mac ou Linux) et te propose la bonne version.
3
Lance l'installateur Sur Windows : double-clique sur le fichier .exe téléchargé. Sur Mac : fais glisser l'app dans ton dossier Applications.
4
Accepte les options par défaut Clique "Suivant" jusqu'à la fin. Coche "Ajouter au PATH" si c'est proposé — c'est utile !
5
Lance VS Code ! Une fenêtre sombre avec un écran d'accueil s'ouvre. Tu y es 🎉
✅ VS Code se met à jour tout seul. Tu n'auras plus jamais à le réinstaller !

Créer un dossier et un fichier

Avant de coder, tu dois ouvrir un dossier de travail dans VS Code. C'est lui qui contiendra tous tes fichiers de projet.

Ouvrir un dossier
1
Fichier → Ouvrir le dossier… Dans la barre de menus en haut, clique sur Fichier puis Ouvrir le dossier.
2
Choisis ou crée un dossier Par exemple : crée un dossier mon-projet sur ton bureau, puis sélectionne-le.
Créer un nouveau fichier
1
Dans l'explorateur, clique sur l'icône 📄+ Passe ta souris sur le nom du dossier dans la barre latérale gauche — une icône "Nouveau fichier" apparaît.
2
Tape le nom du fichier avec son extension Par exemple : index.html ou style.css. L'extension détermine le type de fichier !
3
Appuie sur Entrée Le fichier apparaît dans la liste et s'ouvre automatiquement dans l'éditeur.
⌨️ Raccourci : Ctrl+N pour créer un nouveau fichier, puis Ctrl+S pour l'enregistrer et lui donner un nom.
📄
Les extensions de fichiers

L'extension d'un fichier (la partie après le point dans le nom) indique son type et son langage. VS Code l'utilise pour activer la coloration syntaxique correcte !

Extension Type de fichier Usage
.html HTML Structure des pages web
.css CSS Mise en forme et design des pages
.js JavaScript Interactivité et logique côté navigateur
.py Python Scripts, data science, back-end
.json JSON Données structurées, configuration
.md Markdown Documentation, README de projet
.txt Texte brut Bloc-notes simple, sans mise en forme
mon-projet/ ├── index.html ← la page principale ├── style.css ← les couleurs et styles ├── script.js ← les interactions ├── README.md ← la documentation └── memo.txt ← Note personelle
⚠️ N'oublie jamais l'extension quand tu nommes un fichier ! Sans elle, VS Code (et ton navigateur) ne saura pas quel type de fichier c'est.

L'autocomplétion — ton super pouvoir

VS Code t'aide à écrire plus vite grâce à l'autocomplétion : quand tu commences à taper, il te propose de finir le mot tout seul !

Exemple — tape h1 puis Tab dans un fichier .html :
← Tu tapes : h1 + Tab ← VS Code génère automatiquement : <h1>|</h1>
Exemple — tape ! puis Tab dans un .html vide :
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> <body> </body> </html>
🪄 Le ! suivi de Tab génère toute la structure HTML de base en une seconde. C'est le raccourci Emmet le plus connu !

Zoom sur Live Server

Live Server est une extension qui lance un serveur local et recharge automatiquement ta page dans le navigateur dès que tu sauvegardes ton fichier. Fini le F5 répété !

Comment l'utiliser :
1
Installe l'extension Live Server Cherche "Live Server" de Ritwick Dey dans les extensions et clique Installer.
2
Ouvre ton fichier .html Clique sur ton fichier index.html pour l'activer dans l'éditeur.
3
Clique sur "Go Live" en bas à droite Un bouton Go Live apparaît dans la barre de statut bleue en bas de VS Code.
4
Ta page s'ouvre dans le navigateur ! Maintenant, chaque fois que tu appuies sur Ctrl+S, la page se recharge instantanément.
⚠️ Pour que Live Server fonctionne bien, assure-toi d'avoir ouvert un dossier dans VS Code et pas juste un fichier seul !
