Résumé : CSS en HTML

1. CSS interne

Le CSS interne est défini dans la balise <style> dans le HTML.


<style>
h1 {
    color: blue;
}
</style>

2. CSS inline

Le CSS inline s’applique directement sur un élément HTML.


<h1 style="color: red;">Titre rouge</h1>

3. CSS externe

Le CSS externe est écrit dans un fichier séparé.


/* style.css */
p {
    color: green;
}
        

<link rel="stylesheet" href="style.css">
