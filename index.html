// 02 — engagement

Le club, c'est sérieux

Une séance par semaine, c'est précieux. Pour que tout le monde progresse, le club fonctionne avec des règles claires et respectées.

✅

On vient pour apprendre

Le club est un espace actif. Chaque séance, on avance sur un projet concret. La curiosité est la seule compétence requise.

🤝

On respecte le groupe

Les questions des autres ne freinent pas — elles enrichissent. Perturber la progression du groupe, c'est ne pas respecter le travail de chacun.

⚠️

La motivation, c'est obligatoire

En cas de désengagement répété, l'accès au club peut être suspendu. Pas pour punir — pour protéger l'avancée des élèves qui s'investissent vraiment.

📵

Zéro distraction

YouTube, réseaux, jeux hors sujet — pendant les séances, c'est terminé. L'ordinateur est un outil de création, pas de consommation.

💾

On sauvegarde, on organise

Chaque fichier a sa place, chaque projet son dossier. Apprendre à ranger son travail, c'est aussi une compétence clé.

🚀

On va à son rythme, mais on avance

Pas de compétition. Chaque élève progresse selon son niveau. L'important : ne jamais rester bloqué sans demander de l'aide.

// 03 — parcours

Comment ça se passe

L'élève choisit sa spécialité — page web ou jeux web — car l'environnement et la logique sont différents. Chaque parcours est cohérent de bout en bout.

S01–02

L'ordinateur n'a plus de secrets

Organisation des fichiers, raccourcis clavier, installation d'un éditeur de code, premiers repères dans le terminal.

S05–08

Socle commun — HTML, CSS, JavaScript

Structure d'une page, mise en forme, premières interactions. Les bases partagées avant de choisir sa voie.

S09

🔀 Choix de spécialité

L'élève choisit son parcours selon ses affinités. Deux environnements distincts, deux logiques différentes — chacun avance sur la voie qui lui correspond.

🌐 Page web HTML avancé, CSS responsive, JS interactif. Objectif : une vraie page en ligne, propre et fonctionnelle.
🎮 Jeux web Canvas HTML et JavaScript, boucle de gameplay, un mini-jeu jouable et publié.
S03–04

GitHub — sauvegarder et partager son travail

Créer un compte, comprendre les dépôts, faire ses premiers commits. Chaque projet du club est versionné dès le départ. GitHub devient aussi un portfolio scolaire : l'élève peut partager et réutiliser son code dans d'autres matières, et découvrir concrètement l'utilité de ce qu'il crée.

S21–24

Projet scolaire — coder pour une vraie utilité

L'élève applique ses compétences à un besoin concret : site de révision pour une matière, quiz interactif, mini-jeu éducatif… Le code créé au club devient utile en classe, et GitHub sert à le partager avec ses camarades ou ses profs.

S25+

Projet libre — aller plus loin dans sa spécialité

Chaque élève approfondit sa voie avec un projet ambitieux et un suivi personnalisé. Les travaux sont régulièrement poussés sur GitHub — l'historique des commits reflète la progression réelle.

🌐 Site web moderne Pour les élèves qui souhaitent franchir un cap : Python, Django, base de données. Portfolio, blog, appli web complète.
🎮 Programmation et Algorithmie On consolide les fondations du code : variables, conditions, boucles, structures de données, fonctions. Puis on polish — optimiser, nettoyer, rendre le jeu vraiment jouable et agréable.
S10–20

Faire de sa passion son métier

Si cette univers le passionne et qu'il envisage d'en faire son métier, des événements et des parcours lui seront partagés : portes ouvertes d'écoles, salons, forums d'orientation — des rendez-vous à faire avec ses parents pour explorer les voies qui s'offrent à lui.
