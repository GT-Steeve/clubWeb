# 📘 Support de Cours — HTML, CSS et plus

## 🎯 Objectif du dépôt

Ce dépôt regroupe des supports de cours autour du développement web, notamment :

* HTML
* CSS
* Django
* Et d'autres sujets à venir...

L'objectif est de proposer une approche progressive et accessible pour apprendre à créer, comprendre et améliorer du code.

---

## 🧠 Méthodologie d'apprentissage

La méthode adoptée ici est volontairement différente des approches classiques :

### 1. Priorité aux soft skills

Avant de plonger dans la technique pure, l'accent est mis sur :

* la compréhension globale
* la logique
* la capacité à lire et manipuler du code existant
* l’esprit critique face au code généré

---

### 2. Apprendre à partir de code généré (IA)

L'apprentissage commence par l'utilisation de code généré par IA :

* Observer le code
* Le tester
* Le modifier
* Comprendre son fonctionnement

👉 Objectif : **se familiariser rapidement avec des structures réelles**

---

### 3. Réorganisation et amélioration

Une fois le code compris :

* Le restructurer
* L’optimiser
* Le rendre plus lisible
* Améliorer le style et l’architecture

👉 Objectif : passer du simple usage à une **maîtrise progressive**

---

### 4. Du "Vibe Coding" au code autonome

Le parcours pédagogique suit cette évolution :

1. **Vibe Coding**

   * Manipuler du code généré
   * Expérimenter sans pression
   * Comprendre par la pratique

2. **Appropriation**

   * Modifier en profondeur
   * Corriger et améliorer

3. **Autonomie**

   * Écrire son propre code
   * Structurer ses projets
   * Concevoir des solutions complètes

---

## 🚀 Philosophie

> "Comprendre en manipulant, puis maîtriser en construisant."

L’objectif n’est pas seulement de savoir coder, mais de :

* comprendre ce que l’on fait
* être capable d’améliorer un code existant
* développer une vraie logique de développeur
